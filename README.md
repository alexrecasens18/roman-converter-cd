# Roman Converter CI/CD

This repository contains a Roman Numeral Converter web application developed using:

- HTML
- CSS
- Vanilla JavaScript

The project was created as part of the Software Quality course at Universidad San Jorge.

---

# Features

- Integer to Roman numeral conversion
- Roman numeral to integer conversion
- Roman numeral validation
- Responsive user interface
- Automated functional tests
- Continuous Integration with GitHub Actions
- Continuous Delivery with GitHub Pages

---

# Automated Pipeline

The CI/CD pipeline automatically:

- Executes validation checks on every push
- Runs automated tests
- Verifies project structure
- Deploys the application using GitHub Pages
